<h3>注意事项</h3>
<ol>
<li>开票前20分钟运行软件即可，不要过早运行软件。</li>
<li>自定义开票时间请勿设置过早，一般提前1~2秒即可。</li>
<li>捡漏模式:&nbsp; 指定票档捡漏是指只会检查您选择的票档是否有票，有票则会进行购买。全部票档捡漏是指只要出现有库存的票档，则会自动帮您下单。</li>
